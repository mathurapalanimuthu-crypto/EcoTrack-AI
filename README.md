🌱 EcoTrack AI – Carbon Footprint Tracker

🚀 Overview
EcoTrack AI is a web-based application that helps users track and reduce their daily carbon footprint by analyzing electricity usage, transport, food habits, and shopping behavior.

✨ Features
- Login page with splash screen  
- Home page with dynamic greeting  
- Carbon footprint tracking (electricity, transport, food, shopping)  
- Graph visualization using Chart.js  
- Personalized eco-friendly suggestions
